~ MEMORY ~

Memory to popularna gra polegająca na odnajdowaniu par takich samych kart. Gracz odsłania 2 karty. Jeśli są to takie same karty, zostaną one zdjęte z planszy, jeśli nie, karty będą odwrócone z powrotem. Celem gracza jest zdjęcie wszystkich kart przy możliwie najmniejszej liczbie odsłon. W grze liczona jest ilość błędnych odsłon. Plansza składa się z 42 kart. Powodzenia!
